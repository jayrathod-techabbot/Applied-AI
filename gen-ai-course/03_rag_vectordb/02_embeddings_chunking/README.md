# Embeddings & Chunking

## Introduction

This topic covers the essential techniques for converting text into vector embeddings and splitting documents into optimal chunks for retrieval.

## What You'll Learn

- Understanding text embeddings and their types
- Document chunking strategies
- Choosing the right chunk size
- Handling special content types (code, tables, etc.)

## Key Concepts

1. **Text Embeddings** - Numerical representations of text
2. **Embedding Models** - Models that generate embeddings
3. **Chunking** - Splitting documents into manageable pieces
4. **Overlap** - Maintaining context across chunks

- Basic Python programming
- Understanding of RAG from previous topic
