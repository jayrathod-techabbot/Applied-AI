# Exercise 01: Building an AI Guardrail System

## Task

Create a comprehensive guardrail system for AI applications.

### Requirements

1. **Input Guardrails**
   - Keyword blocking
   - Input validation
   - Rate limiting

2. **Output Guardrails**
   - Content filtering
   - Format validation
   - Sensitive data detection

3. **Monitoring**
   - Log all blocked requests
   - Alert on patterns
   - Dashboard for review

### Deliverable

A Python module with:
- Guardrail classes
- Configuration system
- Logging setup
- Test cases
