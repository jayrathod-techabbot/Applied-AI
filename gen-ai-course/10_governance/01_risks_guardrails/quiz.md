# Quiz: AI Risks & Guardrails

## Questions

### Question 1
What are AI guardrails?

A) Physical barriers around servers
B) Safety mechanisms to prevent harmful outputs
C) Software licenses
D) Network firewalls

### Question 2
What is an "adversarial attack" on AI?

A) A legal challenge
B) Malicious input designed to fool the AI
C) Hardware failure
D) Network downtime

### Question 3
What is model drift?

A) When a model moves physically
B) Performance degradation over time
C) Data corruption
D) Server migration

### Question 4
What is "defense in depth"?

A) Military strategy
B) Multiple layers of security
C) Single firewall protection
D) Data backup only

### Question 5
Which is an input guardrail?

A) Content filtering
B) Keyword blocking
C) Format validation
D) All of the above

### Question 6
What is a hallucination in AI?

A) Visual rendering
B) Generating false but plausible information
C) System crash
D) Data loss

### Question 7
What should fail-safe systems do?

A) Fail completely on error
B) Default to safe state
C) Ignore errors
D) Continue regardless

### Question 8
Why is monitoring important for AI systems?

A) It's not important
B) To detect issues early
C) Only for billing
D) Only for debugging

## Answers

1. B - Safety mechanisms to prevent harmful outputs
2. B - Malicious input designed to fool the AI
3. B - Performance degradation over time
4. B - Multiple layers of security
5. D - All of the above
6. B - Generating false but plausible information
7. B - Default to safe state
8. B - To detect issues early
