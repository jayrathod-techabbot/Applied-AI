# Quiz: Responsible AI

## Questions

### Question 1
What is "demographic parity"?

A) Equal pay for all
B) Equal outcomes across demographic groups
C) Equal hiring
D) Equal training

### Question 2
Which is a core principle of responsible AI?

A) Profit maximization
B) Fairness and non-discrimination
C) Speed only
D) Cost reduction only

### Question 3
What is algorithmic bias?

A) Computer errors
B) Systematic errors in AI that favor certain groups
C) Data loss
D) Network issues

### Question 4
What does transparency in AI mean?

A) Showing all code
B) Making AI decisions understandable
C) Open source only
D) Free access

### Question 5
What is "fairness through awareness"?

A) Ignoring demographics
B) Explicitly considering protected attributes
C) Random selection
D) Speed optimization

### Question 6
Which is a type of bias in AI?

A) Training bias
B) All of the above
C) Data bias
D) Algorithmic bias

### Question 7
What is accountability in AI?

A) No responsibility
B) Clear ownership and oversight
C) Random decisions
D) Ignoring errors

### Question 8
What are "protected attributes"?

A) Hidden features
B) Race, gender, age, etc. (legally protected)
C) Private keys
D) Model weights

## Answers

1. B - Equal outcomes across demographic groups
2. B - Fairness and non-discrimination
3. B - Systematic errors in AI that favor certain groups
4. B - Making AI decisions understandable
5. B - Explicitly considering protected attributes
6. B - All of the above
7. B - Clear ownership and oversight
8. B - Race, gender, age, etc. (legally protected)
