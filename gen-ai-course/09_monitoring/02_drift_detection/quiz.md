# Quiz

## Question 1

What type of drift occurs when the embedding model is updated and vector representations change?

A) Data drift
B) Concept drift
C) Embedding drift
D) Target drift

---

## Answer: C

Embedding drift specifically refers to changes in vector representations due to model updates or changes in the embedding process.

---

## Question 2

Which statistical method is commonly used to measure distribution shift for feature drift detection?

A) Chi-Squared Test
B) Population Stability Index (PSI)
C) T-Test
D) ANOVA

---

## Answer: B

PSI (Population Stability Index) is specifically designed to measure distribution shift and is commonly used in ML monitoring for drift detection.

---

## Question 3

What is "sudden drift" in concept drift?

A) Gradual changes over months
B) Abrupt, immediate changes
C) Periodic patterns
D) Temporary anomalies

---

## Answer: B

Sudden drift refers to abrupt changes in the data distribution, often caused by significant events like policy changes or news events.

---

## Question 4

Which metric would indicate knowledge base staleness?

A) Increased CPU usage
B) Higher error rates
C) Retrieval returning irrelevant results
D) Longer response times

---

## Answer: C

When the knowledge base becomes stale, retrieved documents become irrelevant to current queries, indicating outdated information.
