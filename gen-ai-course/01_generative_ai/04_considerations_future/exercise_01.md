# Exercise 01: Building an AI Ethics Framework

## Task

Create a Python script that implements an AI ethics evaluation framework.

### Requirements

1. **Bias Detection Module**
   - Analyze prompts for potential bias
   - Check outputs for discriminatory content
   - Generate bias reports

2. **Safety Checker**
   - Detect harmful content generation
   - Filter inappropriate requests
   - Log safety concerns

3. **Transparency Generator**
   - Create AI disclosure statements
   - Generate attribution labels
   - Document limitations

4. **Audit Logger**
   - Track AI interactions
   - Record decision rationale
   - Enable compliance reporting

### Deliverable

A Python module with:
- Bias detection functions
- Content safety filters
- Transparency utilities
- Audit logging system
