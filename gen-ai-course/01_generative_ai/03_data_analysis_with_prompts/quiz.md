# Quiz: Data Analysis with Prompts

## Questions

### Question 1
What is a key advantage of using LLMs for data analysis?

A) Always accurate
B) Natural language interaction
C) No limitations
D) Free to use

### Question 2
When analyzing large datasets with LLMs, what should you do?

A) Send all at once
B) Process in chunks
C) Use only first 100 rows
D) Avoid using LLMs

### Question 3
What is a limitation of LLM-based analysis?

A) Cannot understand text
B) Can hallucinate patterns
C) Only works with numbers
D) Cannot categorize

### Question 4
What is "batch processing" in LLM analysis?

A) One prompt for everything
B) Processing data in groups
C) Fast processing only
D) Manual processing

### Question 5
How should you validate LLM analysis results?

A) Never validate
B) Only trust the output
C) Check against known results
D) Ignore errors

### Question 6
Which is NOT a good use case for LLM data analysis?

A) Sentiment analysis
B) Pattern discovery
C) Precise calculations
D) Summarization

### Question 7
What format helps LLMs understand data better?

A) Raw text only
B) Structured formats like tables
C) Images only
D) Audio only

### Question 8
What should you include in analysis prompts?

A) Ambiguous instructions
B) Clear requirements and format
C) No context
D) Single words

## Answers

1. B - Natural language interaction
2. B - Process in chunks
3. B - Can hallucinate patterns
4. B - Processing data in groups
5. C - Check against known results
6. C - Precise calculations
7. B - Structured formats like tables
8. B - Clear requirements and format
