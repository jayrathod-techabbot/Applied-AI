# Exercise 01: Exploring Generative AI

## Task

Create a Python script that demonstrates different aspects of Generative AI:

### Requirements

1. **Basic Interaction**: Create a function that calls an LLM API with a simple prompt
2. **Temperature Experiment**: Compare outputs with different temperature values (0.0, 0.5, 1.0)
3. **Prompt Templates**: Create a reusable prompt template for generating different types of content
4. **Output Parsing**: Parse and display specific parts of the LLM response (tokens, model used)

### Example Prompts to Use

- "What is machine learning?"
- "Explain quantum computing to a 5-year-old"
- "Write a function to calculate factorial"

### Expected Output

Your script should demonstrate:
- API call structure
- Response parsing
- Temperature effect on output
- Error handling for missing API keys

### Bonus

Add support for multiple LLM providers (OpenAI, Anthropic) using a unified interface.
