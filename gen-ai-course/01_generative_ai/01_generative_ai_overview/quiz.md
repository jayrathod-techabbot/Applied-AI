# Quiz: Generative AI Overview

## Questions

### Question 1
What is the main difference between traditional ML and Generative AI?

A) Generative AI can only work with numbers
B) Generative AI creates new content rather than just classifying
C) Generative AI doesn't use data
D) Generative AI doesn't require training

### Question 2
What does "temperature" control in LLM generation?

A) The speed of generation
B) The randomness of output
C) The size of the model
D) The length of output

### Question 3
What is tokenization?

A) Converting tokens to cryptocurrency
B) Breaking text into smaller units for processing
C) Encrypting the input
D) Compressing the data

### Question 4
What is "few-shot learning" in the context of LLMs?

A) Training with very little data
B) Providing examples in the prompt
C) Using a small model
D) Learning quickly

### Question 5
What does the context window of an LLM represent?

A) The size of the model
B) How much text can be processed at once
C) The number of parameters
D) The training data size

### Question 6
Which architecture underlies modern LLMs?

A) CNN
B) RNN
C) Transformer
D) K-Means

### Question 7
What are "parameters" in an LLM?

A) Input settings
B) Learned weights during training
C) API configurations
D) Output formats

### Question 8
What is the primary function of an LLM during inference?

A) Training
B) Predicting the next token
C) Collecting data
D) Testing

## Answers

1. B - Generative AI creates new content rather than just classifying
2. B - Temperature controls the randomness of output
3. B - Tokenization breaks text into smaller units
4. B - Few-shot learning provides examples in the prompt
5. B - Context window is how much text can be processed at once
6. C - Transformer architecture underlies modern LLMs
7. B - Parameters are learned weights during training
8. B - During inference, LLM predicts the next token
