# Exercise 01: Advanced Prompt Engineering

## Task

Create a Python script that implements a prompt engineering framework with multiple techniques.

### Requirements

1. **Implement a prompt builder** that supports:
   - System prompt (role/persona)
   - User prompt (main task)
   - Few-shot examples
   - Output format specifications

2. **Create comparison experiments**:
   - Compare zero-shot vs few-shot performance
   - Test different temperature values
   - Compare with/without chain of thought

3. **Build a prompt templates library**:
   - Classification prompts
   - Extraction prompts
   - Transformation prompts
   - Generation prompts

4. **Implement prompt evaluation**:
   - Measure output quality
   - Compare different approaches

### Example Tasks

- Extract key information from news articles
- Convert informal text to formal
- Generate code comments

### Deliverable

A Python script with:
- Prompt builder class
- Template library
- Comparison runner
- Results display
