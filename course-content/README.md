# Applied AI — From Prompts to Production

> A comprehensive curriculum for developers and engineers who want to build production-grade AI systems using LLMs, agents, RAG, and cloud deployment.

---

## 📚 Course Structure

```
course-content/
├── part-1-foundations/          # Modules 1-2: Generative AI & LangChain
├── part-2-retrieval/            # Module 3: RAG & Vector Databases
├── part-3-agentic-ai/           # Modules 4-6: Agentic Systems, MCP, LangGraph
├── part-4-production/           # Modules 7-10: Architecture, CI/CD, Monitoring, Governance
├── part-5-fine-tuning-deployment/ # Modules 11-13: Fine-Tuning, Deployment, LLMOps
└── part-6-capstone/             # Module 14: Production Projects
```

## 🗺️ Learning Path

```
Fundamentals → Retrieval → Agentic AI → Production Engineering → Fine-Tuning & Deployment → Capstone
  (M1–M2)       (M3)        (M4–M6)           (M7–M10)               (M11–M13)               (M14)
```
